# Smart Event Dashboard

 **[Live Demo](https://nitin386.github.io/event-dashboard/)**

A lightweight browser-based event management app built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies.

## Features

- **Add events** with a title, date, category, and description
- **Event cards** displayed in a responsive two-column grid
- **Mark as completed** — visually fades and locks the card
- **Remove individual events** or clear the entire list
- **Sample event** button to preview the layout instantly
- **Keylogger widget** at the bottom that displays the last key pressed

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Markup | HTML5 |
| Styling | CSS3 (Flexbox, responsive media queries) |
| Logic | Vanilla JavaScript (DOM manipulation, event listeners) |

## Usage

1. Fill in the event title, date, category, and an optional description
2. Click **Add Event** — the card appears in the grid on the right
3. Click **Mark as Completed** on any card to grey it out
4. Click **Remove** to delete a specific event
5. Use **Clear Events** to wipe the list, or **Add Sample Events** to populate a demo card
6. Press any key on your keyboard — the keylogger at the bottom will display it
